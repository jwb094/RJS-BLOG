# Blog Application (React.js)

A responsive blog application built with React.js that demonstrates modern frontend development practices including component-based architecture, dynamic content rendering, search functionality, filtering, and reusable UI components.

This project simulates a real-world blog platform by using dummy data to represent backend API responses, allowing users to browse articles, view post details, filter content, and interact with a structured blog interface.

---

## ✨ Features

- Responsive blog layout using Bootstrap
- Blog homepage displaying featured and latest articles
- Blog post listing page
- Individual blog article details page
- Search functionality for blog posts
- Filter posts by category/tags
- Pagination for blog content
- Reusable blog components
- Dynamic rendering of blog data
- Dummy data used to simulate backend API responses
- Responsive navigation and footer components
- Author and category information display
